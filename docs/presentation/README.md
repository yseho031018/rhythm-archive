# 발표자료 안내

## 최종 평가용

- `presentation.html`: URL로 여는 최종 발표자료
- `final-script-5min.md`: 4분 30초 전체 대본
- `final-speaking-cues.md`: 슬라이드를 읽지 않고 설명하기 위한 말하기 카드
- `demo-script-30s.md`: 30초 시연 영상 촬영 순서
- `final-qa.md`: 예상 질문 답변
- `final-outline.md`: 슬라이드 구성 의도

## 중간 발표 기록

아래 파일은 중간 발표 당시의 별도 보관 자료다. 현재 구현 기술이나 최종 발표 근거로 사용하지 않는다.

- `interim.md`
- `presentation-script-2min.md`
- `qa-prep.md`
- `assets/rhythm-wave-preview.svg`

최종 발표에서는 Flutter, ChangeNotifier, Drift/SQLite, Repository Pattern, 규칙 기반 한 줄 생성이라는 실제 구현만 설명한다.
