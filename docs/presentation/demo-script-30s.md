# 하루톡 30초 시연 영상 계획

## 목표 장면

세 번의 선택이 실제 기록과 생활 패턴으로 연결된다는 점을 30초 안에 보여준다.

## 촬영 순서

| 시간 | 화면 | 동작 | 전달할 의미 |
|---|---|---|---|
| 0–3초 | 기록 시작 | `행복` 선택 | 입력 부담이 작다 |
| 3–7초 | 활동 선택 | `친구` 선택 | 생활 맥락을 함께 기록한다 |
| 7–11초 | 만족도 선택 | `5점` 선택 후 한 줄 생성 | 세 번의 선택으로 기록을 만든다 |
| 11–16초 | 한 줄 결과 | 생성 결과 확인 후 저장 | 결과를 수정·저장할 수 있다 |
| 16–21초 | 한줄 목록 | 방금 저장한 기록 확인 | 저장 흐름이 연결된다 |
| 21–26초 | 감정잔디 | 월간에서 연간 보기로 전환 | 짧은 기록이 장기 감정 흐름으로 누적된다 |
| 26–30초 | 통계 하단 | `토리가 찾은 관계` 표시 | 단순 기록이 생활 회고로 이어진다 |

## 촬영 체크리스트

- 430×900 정도의 모바일 비율로 촬영한다.
- 마우스 이동은 짧고 일정하게 유지한다.
- 화면 전환 후 0.5초 정도 멈춰 결과를 읽을 시간을 준다.
- 음성 없이도 이해되도록 필요한 장면만 보여준다.
- 최종 MP4와 로컬 앱을 모두 준비한다.
- 영상 길이는 28–30초로 맞춘다.

## Windows에서 촬영

1. 최종 공개 앱 URL 또는 로컬 앱을 Chrome에서 연다.
2. 브라우저 창을 모바일 폭으로 맞추고 기록 첫 화면에 둔다.
3. `Win + Alt + R`로 Xbox Game Bar 녹화를 시작한다.
4. 위 표의 순서대로 시연하고 다시 `Win + Alt + R`로 종료한다.
5. `동영상\캡처` 폴더의 MP4를 재생해 길이와 마우스 움직임을 확인한다.
6. 필요하면 앞뒤 여백만 잘라 `harutalk-demo-30s.mp4`로 보관한다.

## 백업 시연 경로

1. Chrome에서 로컬 앱을 미리 연다.
2. 감정잔디와 통계 화면을 각각 다른 탭에 준비한다.
3. 영상 재생에 실패하면 같은 순서를 실시간으로 시연한다.
